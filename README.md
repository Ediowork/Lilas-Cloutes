# Lilas-Cloutes
# Visão geral do projeto
App que controla a roupa no corpo do cliente

Backend
Arquitetura RESTful bem estruturada
Autenticação segura com JWT + Bcrypt
Relacionamentos MongoDB (populate)
Sistema de numeração automática de pedidos (LA000001, LA000002...)
Web
Dashboard com gráficos interativos (Recharts)
Componentes reutilizáveis Material-UI
Tema personalizado nas cores da marca
Interface responsiva (desktop, tablet, mobile)
Mobile
State management com Provider
Carrinho persistente (SharedPreferences)
Navegação fluida e intuitiva
UI/UX moderna e atraente
